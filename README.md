# LoRaWAN-ThreadX
 A STM32WL - LoRaWAN project, based on ThreadX(Azure RTOS).

基于 STM32WLE5CCU6 芯片的 LoRaWAN 终端设备。使用的 RTOS 是 ThreadX(Azure RTOS).



### 一些信息：

- 外部晶振：32.768KHz、32MHz
- region：CN470
- 基础工程的构建软件：STM32 CubeMX
- 硬件地址：https://oshwhub.com/dfasdf/e77-400m22s
- 串口：921600-8-N-1
- LED 灯： PC13（低电平点亮）
